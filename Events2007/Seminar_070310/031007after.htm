達垃斯台工會 2007 年新春首次科技講座

 

達拉斯台工會新春第一次科技講座於三月十日 星期六由一點半至五點半在 Carrollton Josey Ranch Lake Library 舉行，與會人數約五、六十人左右。

 

首先謝慶鏘會長歡迎大家前來參加2007年達拉斯台工會首次講座，他說台工會才成立兩年多，是此地台灣人社團中最年輕資淺的一個，尚有很多進步的空間。不過至今已有三位會員於去年六月、八月、十月間分別回台灣參加伯樂計劃，今天特別請他們來用頭半段時間來分享他們的伯樂經驗，下半段時間則由吳祥發教授講解”資訊科技-創造價值的原動力”。

 

首屆台工會會長張謙益理事第一位上台發表”伯樂計劃—我個人的經驗 ~ 國防科技與民間產業衍生應用的探討”。他先介紹伯樂計劃是一個台灣政府為延攬海外科技人才專案擴增駐點攬才的計劃，因為台灣需要伯樂人才的智慧，激發產業更多創意及商機。台灣需要伯樂人才的統整能力，協助廠商及學研單位，整合研發技術資源。台灣需要伯樂人才的經驗，協助產業走的更穩，更好。伯樂計畫 每年提供25-30名伯樂人才返台短期諮詢服務及參訪相關人才需求單位。目標則是希望透過伯樂計畫，促成海外資深專業人才返國長期服務。張前會長對伯樂計劃延攬海外人才的領域、對象、補助期限及項目都有詳細的解說並透露去年共有26名伯樂人才返台服務，包括達拉斯台工會的張謙益、陳伯勖及葉鴻振三人。

 

接著進入張會長的伯樂計劃主題”國防工業科技與民間科技產業衍生應用的探討”，他說他於2006年 1月初提出申請, 4月初接到通知錄取，同年6月 9日至 7月14日返台服務。因國研院科技政策研究與資訊中心， 有一研究小組正擬研討台灣民間科技及產業衍生於國防工業應用上之可能性，就加入此一國防科技小組共同研究。Qinetiq 是英國的國防科技研發機構，已有一些國防科技衍生於民間應用的實例:例如國防用的偵察地雷系統可在民間用來測量未生嬰兒的心跳；還有國防用的自動異物偵察系統可用於機場安全系統…….等等。至於美國的DARPA – Defense Advanced Research Projects Agency 的Night Vision Technology、Artificial intelligence、 Machine learning ….等等也都應用於民間產業上。在一個多月的研究行程當中，參與推動國防產業發展研討會；與台大商學研究所國防科技管理研究員座談會；參訪過新竹、 中部、 南部科學工業園區；平均一星期兩次與研究小組聚會討論國家關鍵議題，瞭解台灣國防科技工業結構，提供策略建議，以提昇台灣經濟雙嬴的措施。

 

 

服務結束時，張前會長的”國防與民間科技衍生應用之策略建議”內容簡略如下：1.  於研究中心建立平台，做為國科會、國防部、經濟部與民間產業在促進國防工業上的協調溝通管道。2.  建立國防廠商的產能資料庫，提供國防部購案系統分析之用， 此資料庫應由財團法人負責維護。3. 建立機體寓商機於國防工業中，藉以促進國防科技衍生應用於民間產業上。4. 建立有如美國的國安通行證 Security Clearance 體制，俾能讓民間廠商普遍參與機密國防科技研發之工作。最後他發表他的服務感想，肯定伯樂計劃確能從海外引進先進的科技和經驗；伯樂人抱著回饋台灣，為鄉土服務的熱忱， 過程雖然辛苦忙碌，但都覺得充實，很有意義；參訪科學工業園區，留下深刻印象，對台灣科技發展深具信心。至於參於國防科技策略建議之探討心得感想:  瞭解到台灣科技工業的 culture 與美國不同；對民間科技衍生應用於國防發展的潛力， 應繼續發現並鼓勵；看到一批國防科技策略研究員， 不辭艱辛困擾， 仍不斷為台灣的安全及前途努力奮鬥， 令人感佩。

 

第二位的陳伯勖理事，他特地由台灣趕回來參加，卻因臨時感冒不克來演講。第三位是葉鴻振先生，他是台工會剛卸任的前會長，去年十月返台參加伯樂計劃約一個月期間(10月16日至11月17日)，他說伯樂計劃就是將美國經驗帶回台灣，可貢獻的領域很廣 (diversified fields)，不過至少要停留一個月以上。該計畫為服務返國之伯樂人才，於台北國研院成立專案辦公室綜理計畫管理與協調聯繫工作，另外於竹科奈米元件實驗室、南科國家高速網路與計算中心設立分區辦公室，提供伯樂人才返台辦公之用。若有興趣者請洽當地或就近之國科會駐外科技組，或伯樂計畫專案辦公室。葉前會長在他的服務期間共參訪了下列單位：國研院、資策會、竹科管理局、國家高速網路與計算中心、行政院科技顧問組、台電公司電力經濟研究室、經濟部資訊中心、生物醫學開發中心、統一企業資訊中心      、統一企業中央研究所及中部科學園區籌備處。並且在下列單位演講: 統一企業資訊中心、師範大學數學系、國防部海軍司令部、國防部左營營區- 艦隊指揮部及陸戰隊指揮部。

 

伯樂經驗談結束之後，下半場由吳祥發教授演講”資訊科技：創造價值的原動力” Information Technologies：A Powerful Tool Enabling Value Creation)，他說近二十年來，公司的高級經理皆以創造價值為其經營的目標。公司的永續發展，必須以創造價值給顧客及股東為根本，亦即公司的產品有價值，顧客才會購買，因而創造營業收入。有了營業收入，加上有效率的經營，公司才有利潤，從而創造股東的價值。一個有利潤的公司，當然也會連帶提供價值給員工、供應商以及整個社會。

 

至於如何創造價值呢? 答案就是資訊科技。自1990年開始，資訊科技、尤其網路科技，將整個世界偏平化、濃縮化，從而把產品或服務輕而易舉地傾銷於全球市場。

 

利用資訊科技，公司把內部的資訊系統和外部的顧客、供應商及銀行等結合成為一個整體性的組織。這樣的公司，像Wal Mart、Dell、Cisco等，是以高度的效率，取得兢爭上的優勢，從而創造價值。有些公司，認為某些資訊科技非其基本的長處(Core Competency)，因而傾全力於策略上的資訊科技而把非策略性的資訊科技外包(Outsourcing)。不但減低成本，更把接受外包的公司變成公司延伸的一部分，進而提升策略資訊部份的能力，增進兢爭的優勢，創造更大的價值。

 

資訊系統(Information Systems) 以資訊科技為根基，把經營(Management)和運作(Operation)結合為一體。公司的經營程序(Management Process)是擬定作業程序(Operation Process)的計劃(Plan)、執行(Execution)及控制(Control)的評估(Evalution)。作業程序，利用資訊系統以最有效率的方式來控制執行計劃，評估執行的結果。整個資訊系統是一套及時性(Real Time)迴控(Feedback)系統。公司的經營者能及時性去瞭解而掌握其負責的作業程序。這就是效率的來源。今天美國很多公司能在全球佔有兢爭上的優勢，便是靠這Real Time Feedback

資訊系統的運作。

 

利用資訊系統，把整個作業程序革命性地改變，即所謂作業工程的再造(Process Reengineering)也能提高作業的效率(Efficiency and Effectiveness)，從而提高兢爭上的優勢，創造產品的價值。有了效率，公司就可以成本的領導性(Cost Leadership)或產品的特異性(Product Differentiation)來兢爭。當然最好的是佔有性的藍海策略下的市場，這個策略是從新擬定市場的性質而取得該市場。整個市場就像沒被鮮血染紅的海，蔚藍一片，沒有廝殺的痕跡，完完全全的佔有，蘋果(Apple)公司的IPod是一個好例子。

 

總而言之，一個公司要替顧客及股東創造價值才能生存。要創造價值必須有兢爭或創價的策略，策略的實現又必以資訊科技來支持而成。

 

吳教授一付授課姿態來講解，聽眾專心聽講，踴躍地發表意見。時光飛逝，五點半已到，只好等有機會下次續攤，台工會春季科技講座就此結束。在此特地感謝李紫鳳、陳秀惠及孫佳惠三位辛勞準備多樣水果、餅乾及茶水等款待大家。
